Easiest way of redirection from page or post using shortcode.
== Description ==

<strong>Easy Redirect</strong> can be used to make redirections possible from a page or post.<br />We can use this plugin for redirections in six different ways:<br />
1. Redirection After a Time<br />
2. Redirection With Countdown<br />
3. Redirection With Countdown Delay<br />
4. Redirection With Popup<br />
5. Redirection With Popup Delay<br />
6. Redirection With Popup and Countdown<br />

For a demo or instructions for use, you can check our <a href="http://demo.rhygel.com/wordpress/easy-redirect">Demo Page</a>.

<strong>Use of other Shortcodes</strong><br />
We can also use other shortcodes within easy redirect shortcode. It could be used in two ways:<br />
1. Using shortcode in URL attributes.<br />
2. Using shortcodes within popup content.For instructions please <a href="http://demo.rhygel.com/wordpress/easy-redirect#shortcode-use" target="_default">Check Here</a><br />


For any support or ideas you can raise a ticket at <a href="http://rhygel.freshdesk.com">Support Portal</a> or can send us a message at <a href="https://facebook.com/rhygelinc">Facebook</a>.

== Installation ==


1. Go to Plugins › Add New.
2. Search for *Easy Redirect*.
3. Look for this plugin, download and activate it.
4. Write short code on page for redirect.

== Frequently Asked Questions ==

= How to use Easy Redirect =

<br />
You can add our shortcode anywhere at a page or post and it will magically work. For demo and using instruction you can check our <a href="http://demo.rhygel.com/wordpress/easy-redirect">demo page</a>.
